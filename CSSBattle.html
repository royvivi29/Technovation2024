Rule & Regulation

1. The event is open to all participants from the college.

2. Participants can be beginners or individuals with basic knowledge of HTML and CSS.

3. The event is individual based; each participant competes independently.

4. Each participant must have a personal laptop for the event.

5. Visual Studio Code (VS Code) is the recommended code editor; participants are required to have it installed on their laptops.

6. Mobile devices will not be allowed for code submission.

7. Preliminary rounds will be conducted before 9th February.

8. Challenges will cover various CSS properties and their applications.

9. Emphasis will be on learning, and detailed descriptions will be provided to enhance participants' understanding.

10. All code submissions must be made within the specified time limit.

About The Events

Preliminary rounds:

• Before every round, a detailed description of the challenge will be provided to ensure that beginners can learn and grasp the concepts.

• Participants will be provided with HTML files, and they need to modify the CSS to achieve the specified outcomes.

• A set number of participants with the best solutions will advance to the final round.

Finals:

• Finalists will compete in a series of 1v1 challenges.

• Each finalist will be given a unique challenge to solve within a time limit.

• Before each final round, a description will be given to guide participants in their approach.
